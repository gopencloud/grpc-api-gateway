# Installation

This fork currently offers only build from source. Binaries, maybe, will be later.

## Install from Source

```sh
go install github.com/gopencloud/grpc-api-gateway/codegen/cmd/protoc-gen-openapiv3@<version>
go install github.com/gopencloud/grpc-api-gateway/codegen/cmd/protoc-gen-grpc-api-gateway@<version>
```

To retrieve the latest version, replace `<version>` with `latest`.
